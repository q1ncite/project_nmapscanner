# Simple nmap scanner

## How to install:
```bash
pip3 install python3-nmap
git clone https://github.com/q1ncite/project_nmapscanner
cd project_nmapscanner
python3 scanner.py
```

## How to use:
Enter one ip addresses and just wait.
